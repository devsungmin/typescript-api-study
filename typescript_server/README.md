### Started Example Project
```
docker을 사용하지 않고 실행 할 경우 아래 명령어를 사용해주세요!
```
1. Modify .enf-sample file
2. `vim ./.env-sample`
3. `mv ./.env-sample ./.env`
4. install node_modules package
   ```
   $ npm install
   ```
5. start project
   ```
   $ npm start
   ```
